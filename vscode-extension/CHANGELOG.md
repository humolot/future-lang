# Changelog

## 0.4.3 (2026-06-13)

Initial release.

- Syntax highlighting for all Future keywords, namespaces, string interpolation, and comments
- Code snippets for every language construct (`if`, `fn`, `for`, `while`, `try`, `on`, `every`, `stream`, `agent`, `use`, AI, HTTP, assert, memory, etc.)
- Language configuration: `#` line comments, bracket matching, auto-indent on `if/function/for/…`, de-indent on `end/else/catch`
